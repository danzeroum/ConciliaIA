# 🎓 BuildToValue v7.0 - Onboarding de Novos Integrantes

## 🎯 Objetivo
Acelerar a curva de aprendizado de novos membros garantindo entendimento das squads IA, orquestração e processos de entrega.

## 🗓️ Dia 1
- **Environment Setup:** instalar pré-requisitos, clonar repositório, executar `./scripts/dev-setup.sh`, validar serviços.
- **Architecture Overview:** ler `README.md`, `docs/ARCHITECTURE.md`, revisar personas e fluxo de orquestração.
- **Primeira Contribuição:** escolher um `good first issue`, implementar correção e abrir PR seguindo o template.

## 📅 Semana 1
- **Deep Dives:**
  - Smart Router
  - Auto-RAG Engine
  - Confidence Tracking System
  - Ethics Guardian
- **Hands-on:** adicionar nova persona, ajustar activation matrix, capturar lesson learned, gerar ADR.

## 📆 Mês 1
- **Tópicos Avançados:**
  - Performance tuning
  - Segurança e governança
  - Multi-região e DR
  - Troubleshooting em produção
- **Contribuição:** liderar uma feature média, participar de discussões arquiteturais, mentorar novo integrante.

## ✅ Checkpoints
- [ ] Ambiente local operacional.
- [ ] Entendimento dos níveis de autonomia das IAs.
- [ ] Familiaridade com Decision Ledger e dashboards.
- [ ] Primeiro PR aprovado e mergeado.
- [ ] Participação em pelo menos uma cerimônia de arquitetura.
