# 🧰 BuildToValue v7.0 - Stack Tecnológica

## 🗂️ Linguagens
- **Primárias:** Java 17+ (Spring Boot 3.x), Python 3.11+ (Orquestração/ML)
- **Secundárias:** TypeScript/JavaScript (Frontend), Shell/Bash (Automação)

## 🏛️ Frameworks
- **Backend:** Spring Boot 3.2+, Spring Data JPA, Spring Security OAuth2
- **Orquestração:** LangChain / LangGraph + camada customizada
- **Frontend:** React 18+, Tailwind CSS, Recharts

## 🗄️ Bancos e Persistência
- PostgreSQL 15+
- ChromaDB (vetores)
- Redis 7+ (cache)
- Prometheus (métricas/time series)

## 🤖 Modelos e ML
- **LLMs:** OpenAI (GPT-4/Turbo), Anthropic (Claude 3 Opus/Sonnet), Google Gemini Pro
- **Embeddings:** `sentence-transformers/all-MiniLM-L6-v2`
- **Frameworks:** scikit-learn, TensorFlow, PyTorch

## 🔍 Observabilidade
- Tracing: OpenTelemetry + Jaeger
- Métricas: Prometheus + Grafana
- Logging: JSON estruturado + ELK opcional

## ⚙️ Infraestrutura
- **Orquestração:** Docker / Docker Compose / Kubernetes
- **IaC:** Terraform + Ansible
- **CI/CD:** GitHub Actions, GitLab CI, Jenkins (suporte)

## ✅ Testes
- Unit: JUnit 5, pytest
- Integração: Testcontainers, WireMock
- Performance: k6, Apache JMeter
- Segurança: OWASP Dependency Check, Snyk, Trivy
